    Microsoft Office 365

  • Creating A New User

    All Office 365 users are created when a new user is created in Active Directory.

    This is made possible by an ADSync tool which runs every 30 minutes on the SIGMAPSYNC server.

    !! Users can not be created in Office 365 portal, if attempted you will receive an error. !!

  • Assigning & Removing Licenses

    Office 365 accounts require licenses to access any account app or data. By default no license is assigned upon account creation.

    To assign or remove a license to an account follow the procedure below:

    - Go to the Office 365 Dashboard

    - Click on the Admin app.

    - Search for the user you wish to add or remove a license for.

    - Go to the Licenses tab and assign or remove the appropriate license required for the user.

  • Downloading & Installing Microsoft Office Apps

    Certain Office 365 licenses include access to desktop Office apps.

    To download and install Office apps to a device please follow the procedure below:

    - Go to the Office 365 Dashboard

    - Login with Office 365 account credentials

    - Click on the Install Office button in the upper right corner and select Install office Apps

    - All Office apps will download and install automatically

  • Updating Office Apps

    Microsoft Office 365 apps are scheduled to be updated on a monthly basis automatically.

    To manually download and install Office app updates please follow the procedure below:

    - In any Office app go to File > Office Account > Office Updates > Update Now

    - Office will check if any updates are available and download them automatically

    - All Office apps must be installed for update to complete

  • Converting Mailboxes to other Mailbox Types

    There are several types of mailboxes in Office 365

    Regular - Standard User Mailbox (License Required)

    Shared - Mailbox that can be shared between one or more individuals (Free)
